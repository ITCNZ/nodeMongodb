# nodeMongodb

> node  简单案例，一个文章的增删改查案例，一个爬虫案例

> 首先要确保安装了MongoDB, cd 到mongodb/bin目录下，执行“./mongod”命令，启动MongoDB。

## Build Setup

``` bash
# install dependencies
npm install

# 文章增删改查 启动
models 目录下 —— supervisor listrouter.js

# 爬虫案例
server 目录下 —— supervisor nodespider.js

# Tips
实时监控先安装 全局supervisor
```
